# NSPAQ-Sms-Bomber
Bu proje, Node.js tabanlı, terminal üzerinden çalışan interaktif bir SMS gönderim arayüzüdür. Modüler yapısı ve görsel animasyonları ile kullanıcı dostu bir deneyim sunar.

## ✨ Özellikler

* **Modüler Mimari:** JavaScript dosyaları (`modules/`) ile kolayca geliştirilebilir yapı.
* **Görsel Arayüz:** `animations.js` ve `ascii.js` ile zenginleştirilmiş terminal görselleri.
* **Hızlı ve Hafif:** Node.js motoru üzerinde minimum kaynak tüketimi.

## 🛠️ Kurulum

Projeyi yerel makinenizde çalıştırmak için şu adımları izleyin:

1.  **Projeyi Klonlayın:**
    ```bash
    git clone https://github.com/nspaqsec/NSPAQ-Sms-Bomber.git
    ```

2.  **Proje Klasörüne Gidin:**
    ```bash
    cd NSPAQ-Sms-Bomber
    ```

3.  **Gerekli paketleri yükleyin:**
    ```bash
    npm install
    ```

4.  **Başlatın:**
    ```bash
    node index.js
    ```
